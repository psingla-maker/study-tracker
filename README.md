# Study Tracker (Java)

## Overview
This is a simple Java console program that tracks daily study hours over a 7-day period and calculates the total weekly study time.

## How it works
- The program asks the user to input hours studied for each day (1–7)
- It stores and adds the values
- It outputs the total study hours for the week

## Technologies Used
- Java
- Scanner (for user input)

## Purpose
This project was created to practice basic Java programming concepts such as loops, user input, and arithmetic operations.

## What I learned
- Using loops in Java
- Handling user input
- Basic program structure in Java
